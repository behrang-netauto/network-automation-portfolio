# 02 Lambda Anonymous Function

This folder contains sorting exercises based on network-style data.

## Purpose

The purpose of this folder is to practice `lambda` functions, custom sort keys, and simple parsing patterns using examples that look like network data.

## What it shows

- sorting interface names by slot and port
- sorting VLAN names numerically
- sorting ACL lines by sequence number
- sorting IP addresses by octet values
- sorting route prefixes
- extracting prefixes from route-like and BGP-like text

## Type of exercises

The scripts focus on `sorted()` and custom key functions. The input data is intentionally network-flavored so the Python examples stay close to later automation work.

## Role in the repository

This folder is a compact Python practice area. It helps build confidence with sorting and data normalization before moving into inventory handling, command output parsing, and generated artifacts.
