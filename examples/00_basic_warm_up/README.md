# 00 Basic Warm-Up

This folder contains small Python warm-up exercises with light network-oriented examples.

## Purpose

The purpose of this folder is to practice basic Python syntax and text-processing patterns before moving into larger network automation workflows.

## What it shows

- basic Python data types and naming
- string formatting
- file reading
- simple filtering
- regular expressions
- generator functions with `yield`
- basic error handling around file access
- simple log anonymization

## Type of exercises

The scripts are intentionally small and progressive. They start with basic Python examples and then move toward network-flavored tasks such as reading IP addresses, parsing interface output, extracting router model information, filtering syslog lines, and masking IP addresses in logs.

## Role in the repository

This is a learning foundation folder. It is not intended to be a polished automation project. It provides background practice for the more practical network automation projects later in the repository.
